# Chiikawa Pocket 中文站

可爱的口袋吉伊卡哇游戏攻略站

## 项目介绍

Chiikawa Pocket中文站是一个专注于提供"可爱的口袋吉伊卡哇游戏"全面攻略的网站。

- 游戏攻略
- 图片库
- Chiikawa小游戏


## 安装

```bash
$ npm i 
```

## 本地开发

```bash
$ npm run start
```

此命令启动本地开发服务器并打开浏览器窗口。大多数更改无需重启服务器即可实时显示。

## 构建

```bash
$ npm run build
```

此命令将静态内容生成到`build`目录中，可以使用任何静态内容托管服务进行部署。
